Explanation

In it's most basic configuration, Mapbuilder can be used to simply put a map in a web page.
The configuration for this page consists of a single model (a Web Map Context document) and one widget (a MapPane). See the Interactive Map Viewer to for an example of an interactive map, as well as some of the more common ways to work with MapBuilder.
Have a look at this guide if you want to modify this example.
