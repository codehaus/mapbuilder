MapBuilder WMTS Map Viewer

Explanation

In it's almost most basic configuration, Mapbuilder can also be used to simply put a WMTS map plus an overview map in a web page.
The configuration for this page consists of a single model (a OWS Context document) and three widgets (MapPaneOL for the map, PanZoomBar for OpenLayers controls and OverviewMap). See the Interactive Map Viewer to for an example of an interactive map, as well as some of the more common ways to work with MapBuilder.
