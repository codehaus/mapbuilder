Explanation

This demo shows how to combine different (opaque) map layers, in a google maps style. To do this you need to have a collection list with the WMCs you want to use. Note that the WMCs in the list can contain multiple layers.
When you click on one of the three buttons on top of the map it will load the WMC corresponding with that particular button and it will use the current extent. This way you can easily switch between a map and satellite view for instance.
