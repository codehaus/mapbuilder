Explanation

This demo provides a minimal interactivity to a map. It includes buttons to zoom and pan. Select a tool and click and drag on the map. To add or remove functionality to the map one has to edit config.xml. See the User Guide for more information on this.
Apart from the buttons there are three examples on how to interact with the map from within a webpage.
Have a look at this guide if you want to modify this example.

Examples of working with MapBuilder:

  • Load a different map by a URL parameter
  • Load a different map by JavaScript function call
  • Load the map with a different Bounding Box
