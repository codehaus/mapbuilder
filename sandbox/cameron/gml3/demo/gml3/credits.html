MapBuilder Open Source GML3 Demo for OGC's OWS4 testbed.

About

The Open Source GML Viewer renders GML3 and WMS layers as described by an OWS Context document within a Web Browser.

The Viewer can be distributed with a GML dataset and requires no more than a recent Web Browser to run.

Testing has been done on both Firefox 1.5 and Internet Explorer 6.

Vector rendering is achieved using SVG in firefox and VML in Internet Explorer.
