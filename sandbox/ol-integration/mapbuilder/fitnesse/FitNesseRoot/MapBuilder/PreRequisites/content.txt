!2 Introduction
Mapbuilder relies on fitnesse http://www.fitnesse.org for its acceptance tests.
A !-RubyFitServer-! is included in the repository

 * Required Packages to be independantly installed on the test machine:
  * Ruby 1.8.4 or higher http://rubyforge.org/projects/ruby
  * !-RubyGems-! (nice to have) http://rubyforge.org/projects/rubygems/
  * WATIR http://wtr.rubyforge.org/
   * Make sure that you do not have 2 copies of Watir installed. in ruby/lib/ruby/site_ruby/1.8/watir and ruby/lib/ruby/gems/1.8/watir-1.4.1 (if you have use gem to install watir (from command: gem install watir)
  * !-AutoIt-! http://www.hiddensoft.com/AutoIt/
