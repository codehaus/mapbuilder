CGDI Interoperability Pilot, Browser Clients

Geobase Discovery/Browse/Download
Allows users to search the GeoBase Catalog for map layers or zoom to a placename, build a map from the layers, then save the map as an OWS Context document, or a layer as GML.
User Feedback
Users can view features, select and feature, then edit it to suggest a change. The suggested change is submitted to a feedback feed, where data custodians can review and apply the suggested change.
Feedback Feed
Allows provence data custodians to monitor user update suggestions.
Update Feed
Allows users to view changes made to Provence databases.
Emergency Management Event Feed
Allows users to monitor emergency management events.
