CGDI Interoperability Pilot, Browser Clients

CGDI-IP User Feedback Form
Users can view features, select and feature, then edit it to suggest a change. The suggested change is submitted to a feedback feed, where data custodians can review and apply the suggested change.
CGDI-IP Emergency Management Notification
Allows users to view the feedback feed.
CGDI-IP Geobase Discovery/Browse/Download
Allows users to search a Catalog for map layers, build a map from the layers, then save the map, or the layers.
