MapBuilder Ship Track Mapbuilder Demo using geoRss Feeds

Explanation

This shows an example of a GeoRSS feeds being displayed, containing a line and point geometry.

It also shows how popups, a listing and legend are displayed from the GeoRSS. Note that the GeoRSS feed is transformed to a FeatureCollection, rendered using GmlRendererOL, and styled by SLD.
