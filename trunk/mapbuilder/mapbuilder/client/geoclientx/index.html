GeoClientX web map browser

GeoClientX, an upcoming addition to the GeoServer project, is a web map browser based on the Open GIS Consortium's Web Map Service and Web Feature Service specifications.  This initial draft has a number of features only roughed in.

View the Greater Victoria Green Map. This demonstration map includes background information from a web map service (WMS) and points from a web feature service (WFS) layered on top (the points will be loaded after you zoom in--hold your mouse over a point to see attribute information).

Or connect to another web map service. Optionally, you can also connect to a web feature service, which must be in the same spatial reference system (projection) as the web map service. Note that a WFS can be added only if a WMS is also loaded and that only point-type features are supported.  The default WMS address given is for a map of Canada produced by DMSolutions.
