Explanation

Flickr Demo using the Community MapBuilder Flickr Feed
Note: Requires Server/proxy support to access Flickr (Assuming it is up). It takes a few minutes to load the pictures from Flickr...
The demo parses the GeoRSS feed and displays the content of the feed at their correct location. It uses SLD documents to parse the GeoRSS and render it as vectors on the map.
Note due to an API/Key change this demo no longer works. Once we have secured a new key it will be enabled again.
