Mapbuilder Demo Pages

Browsers supported are Internet Explorer 6 or later, Mozilla 1.3 or later, Netscape 6 or later.

Simple Map Viewer
A minimal map viewer which allows a user to view layers from Web Map Services (WMS) as configured by a Web Map Context (WMC) document which describes layers to view. This is a good demo to copy when building a new application.
Status: beta
Demis Maps Viewer
A Demis maps viewer which allows a user to switch maps from various Web Map Services (WMS) as configured by a Web Map Context (WMC) document which describes layers to view.
Status: beta
WMS Time Series
Shows maps as a time series. Zoom in to a region of interest.
Status: alpha
Complete configuration
Not pretty to look at but shows integration of most components. Intended for testing extensions to the code base.
Status: alpha
Requirements:
  • Some functionality requires the server proxy and serialize scripts in the configuration file. Implementations of these are available for PHP enabled servers and Java Servlet Container (like Tomcat).
Feature Entry
Shows editing and updating of GML features and uploading to a Transactional Web Feature Service (WFS-T).
Status: prototype
Requirements:
  • Mapbuilder needs to be installed in a Java Servlet container (like Tomcat) in order to upload events to a WFS-T
  • This demo accesses data from a default Geoserver installation which should be installed in the same Java Servlet Webapps directory as Mapbuilder.
For more details, refer to the Installation Instructions.
Context Editor
Creation and editing of Web Map Context documents from searching and browsing a registry.
Status: prototype
Requirements:
  • Some functionality requires the server proxy and serialize scripts in the configuration file. Implementations of these are available for PHP enabled servers and Java Servlet Container (like Tomcat).
geoRSS
A small application demonstrated at OSG'05 that showed how to extend Mapbuilder. This one adds a GmlPointRenderer to the suite of MapContainer widgets. Also this variation of it.
Status: done
