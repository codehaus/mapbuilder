Map tools

A map can also have some interactive functionality by adding tools to the MapPane widget.

In this configuration This map also has an AoiBox tool to display an Area of Interest (AOI) for the model. The AOI can either be a rectangular bounding box or a single point. The colour, line width and size of point display are customized in the config file.

A tool is also available to handle mouse events on the MapPane widget. Click, or click and drag out an AOI box on this map. In the next sample page, these mouse events will be used to control zoom in and out for the map.

This tutorial page also demostrates that any number of models can be added to the same page.
